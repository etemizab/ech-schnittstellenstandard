## Anhang C – Abkürzungen und Glossar · Version: ![v1.0.0](https://img.shields.io/badge/version-1.0.0-blue)

### Abkürzungen

| Abkürzung | Bedeutung |
|-----------|-----------|
| **AHV** | Alters- und Hinterlassenenversicherung |
| **BFS** | Bundesamt für Statistik |
| **RfC** | Request for Change – Änderungsantrag |
| **RHG** | Bundesgesetz über die Harmonisierung der Einwohnerregister und anderer amtlicher Personenregister |
| **VZG** | Bundesgesetz über die eidgenössische Volkszählung |

---

### Glossar

| Begriff | Erklärung |
|---------|-----------|
| **Ereignis** | Das Eintreten eines spezifischen Sachverhalts, z.B. Geburt oder das Erreichen eines bestimmten Zeitpunkts wie Volljährigkeit. |
| **Ereignismeldung** | Meldung aller relevanten Informationen zu einem bestimmten Meldegrund an eine oder mehrere externe Stellen. |
| **Meldegrund** | Ein Ereignis, das eine Datenmutation notwendig macht und eine Meldung an Umsysteme auslöst. |
