### 3.3.8.1 personIdentification – Personenidentifikatoren · Version: ![v1.0.0](https://img.shields.io/badge/version-1.0.0-blue)

Es können entweder die Identifikatoren zu einer natürlichen Person oder einer Unternehmung geliefert werden.

**Austauschformat:**

- eCH-0044:personIdentificationType  
- eCH-0044:personIdentificationLightType  
- eCH-0011:partnerIdOrganisationType
